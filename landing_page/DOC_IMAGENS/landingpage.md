Crie uma landing page institucional premium para o Instituto Plantaê, uma organização de ecologia, sustentabilidade, educação ambiental e restauração ecológica.

A landing page deve seguir exatamente a estética da referência enviada: visual botânico sofisticado, fundo verde escuro profundo, tons creme, dourado/amarelo inspirado no ipê-amarelo, tipografia editorial elegante, imagens realistas de natureza, composição refinada, espaçamento generoso e aparência de site profissional de alto padrão. Não quero aparência genérica de site feito por IA, nem cards comuns com ícones padrão.

TECNOLOGIAS OBRIGATÓRIAS:
- Next.js com React
- TypeScript
- Tailwind CSS
- GSAP com ScrollTrigger
- Lenis para smooth scroll
- Framer Motion apenas se fizer sentido para microinterações
- Componentização limpa
- Responsivo mobile-first
- Código pronto para produção
- Performance otimizada
- Imagens com next/image
- Animações suaves, elegantes e não exageradas

REQUISITOS DE DESIGN:
- Layout extremamente profissional.
- Não usar cards genéricos coloridos demais.
- Não usar ícones de pacote visual comum como se fossem placeholders.
- Criar ou simular ícones próprios em SVG line-art com tema botânico.
- Usar imagens naturais premium: ipê amarelo, folhas, mãos segurando muda, floresta, viveiro, raízes, texturas de folha.
- Usar elementos decorativos sutis: folhas em baixa opacidade, sombras orgânicas, molduras em arco, bordas finas douradas, texturas leves.
- O site deve parecer de uma marca real, não template.
- Estética: luxo natural, editorial, institucional, ambiental, moderno e acolhedor.

ANIMAÇÕES COM GSAP:
Implementar:
- Smooth scroll com Lenis.
- ScrollTrigger para revelar seções.
- Parallax suave em imagens botânicas.
- Split text no título principal.
- Header com mudança de estado no scroll.
- Cards com reveal em stagger.
- Ícones com animação de desenho SVG.
- Count-up nos números de impacto.
- Transições suaves entre seções.
- Microinterações nos botões com hover elegante.
- Evitar animações exageradas. Tudo deve ser premium, lento, fluido e natural.

RESPONSIVIDADE:
- Desktop: hero em duas colunas, imagem grande à direita.
- Tablet: reduzir escala da árvore e manter boa hierarquia.
- Mobile: hero em coluna, texto primeiro, imagem depois, menu hambúrguer, cards em uma coluna.
- Garantir excelente legibilidade em telas pequenas.
- Espaçamentos fluidos usando clamp().
- Imagens otimizadas e sem quebrar layout.
