## O que entregar

### 1. Setup do projeto
- `go.mod` com deps: chi v5, pgx v5, golang-migrate v4, validator/v10, golang-jwt/jwt/v5, google/uuid, golang.org/x/crypto, aws-sdk-go-v2
- `cmd/server/main.go` — entrypoint esqueleto (config → DB → router → ListenAndServe)
- `docker-compose.yml` — PostGIS 16-3.4 (porta 5432) + MinIO (portas 9000/9001)
- `Makefile` — targets: run, migrate-up, migrate-down, migrate-create
- `.env.example` — todas as env vars
- `.gitignore`

### 2. Migrations (todas as tabelas do DATABASE.md)
`migrations/` com golang-migrate. 11 migrations (up + down):
enable_postgis → organizations → users → refresh_tokens → asset_types → assets → media → manejos → monitoramentos → audit_logs → idempotency_keys

### 3. Config
- `internal/config/config.go` — MustLoad() que lê env vars e faz log.Fatal se obrigatória faltar

### 4. Shared packages
- `internal/shared/apperror/errors.go` — AppError + constructors (NotFound, Conflict, Forbidden, InvalidStatusTransition, Validation)
- `internal/shared/response/json.go` — JSON(), HandleError(), BadRequest(), ValidationError()
- `internal/shared/pagination/cursor.go` — ParseCursorParams()
- `internal/shared/constants.go` — Status, Roles, HealthStatus, AuditActions
- `internal/shared/context.go` — GetUserID(ctx), GetOrgID(ctx), GetRole(ctx)

### 5. Middleware base
- `internal/middleware/request_id.go` — UUID no X-Request-Id + context
- `internal/middleware/logging.go` — slog JSON (method, path, status, duration, request_id)
- `internal/middleware/recover.go` — panic → 500 + log
- `internal/middleware/security_headers.go` — nosniff, DENY, HSTS, CSP
- `internal/middleware/cors.go` — origens do config
