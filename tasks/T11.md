## TDD — Obrigatório

### Suposições
1. O backend (T08) está servindo endpoints públicos em `/api/v1/public/*`.
2. O dispositivo possui suporte a `expo-location` e `expo-camera`.
3. Imagens são servidas via presigned URLs do MinIO/S3.

### Testes
Os testes foram realizados nas camadas de API e Queries para garantir a integridade dos dados:
- `mobile/src/features/public/__tests__/api.test.ts` (Validando endpoints e transformações)
- `mobile/src/features/public/__tests__/queries.test.ts` (Validando estados de loading, error e cache do TanStack Query)

### Implementação
A implementação seguiu o padrão de design "Eco Minimalist Glassmorphism":
1. **Mapa**: Uso de `react-native-maps` + `supercluster` para alta performance.
2. **UI/UX**: Refatoração sênior utilizando `useSafeAreaInsets` para adaptabilidade total.
3. **Hardware**: Permissões gerenciadas via Expo para Câmera e Localização.
