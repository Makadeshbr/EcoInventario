## O que entregar

### 1. Mapa Explorar
- `app/(guest)/(map)/index.tsx`
- Mapa interativo (react-native-maps ou mapbox)
- Carrega assets da API pública: GET /public/assets?bounds=...
- Markers nos assets. Clusters para alta densidade.
- Filtro por tipo de asset (chips, dados de GET /public/asset-types)
- Mostra localização do usuário (expo-location)
- Tap no marker → navega para ficha pública

### 2. Ficha Pública do Asset
- `app/(guest)/(map)/asset/[id].tsx`
- Carrega GET /public/assets/{id}
- Exibe: tipo, galeria de fotos (swipeable), mapa com marker
- Seção "Manejos": lista com descrição + fotos before/after
- Seção "Monitoramentos": lista com health_status (badge colorido) + notas
- Botão "Como chegar" → abre app de mapas nativo (Linking)
- Sem dados internos (sem created_by, sem status, sem rejection_reason)

### 3. Scanner QR (visitante)
- `app/(guest)/(scanner)/index.tsx`
- Câmera com expo-camera + leitura de QR
- Ao ler QR → chama GET /public/assets/resolve-qr?code=...
- Se is_available=true → navega para ficha pública
- Se false → mensagem "Ativo não disponível para visualização"

### 4. Sobre
- `app/(guest)/(about)/index.tsx`
- O que é o EcoInventário, quem mantém, contato
- Botão "Sou profissional → Entrar" → navega para (auth)/login
- Versão do app
