## O que entregar

### 1. Usuários CRUD (ADMIN only)
- `/dashboard/users/page.tsx`
- Tabela: nome, email, role (badge), ativo (toggle), data criação
- Botão "Novo usuário" → modal/drawer com form (nome, email, senha, role)
- Editar inline ou modal: alterar role, ativar/desativar
- Não pode deletar a si mesmo

### 2. Tipos de Asset CRUD (ADMIN only)
- `/dashboard/asset-types/page.tsx`
- Tabela: nome, descrição, ativo (toggle)
- Botão "Novo tipo" → modal com form
- Editar inline ou modal
- Desativar (toggle is_active, não delete)

### 3. Auditoria (ADMIN only)
- `/dashboard/audit/page.tsx`
- Tabela de audit logs: ação, entidade, quem, quando
- Filtros: entity_type, action, performed_by, período (date range)
- Expandir row: mostrar campo `changes` formatado como diff visual (campo → old → new)
- Paginação cursor-based

### 4. Mapa Público Web
- `/public/page.tsx` (fora do /dashboard, sem auth)
- Mapa Leaflet com assets approved only (GET /public/assets?bounds=...)
- Mesmo visual do mapa visitante mobile mas em formato web
- Click no marker → popup com resumo + link para detalhes
- Detalhes inline ou page: GET /public/assets/{id}
- Cache-friendly (dados públicos, CDN-ready)
